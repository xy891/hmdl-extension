# 喜马拉雅下载器 (Ximalaya Downloader)

一款强大的喜马拉雅音频下载 Chrome 扩展程序，专门用于批量下载您已购买的音频内容。

![截图](docs/screenshot.png)

## 主要功能

- **一键批量下载**：自动滚动并抓取专辑下所有音频。
- **智能去重**：通过扫描本地文件列表，自动跳过已下载的内容，节省流量和时间。
- **自动翻页**：支持多页专辑的自动化连续下载。
- **精美 UI**：内置简洁直观的操作界面和使用指南。

## 如何使用

### 1. 安装扩展
- 方法 A：从 Chrome Web Store 安装（推荐）。
- 方法 B：开发者模式加载：
  1. 下载本项目并解压。
  2. 打开 Chrome 扩展程序页面 (`chrome://extensions/`)。
  3. 开启“开发者模式”。
  4. 点击“加载已解压的扩展程序”，选择项目中的 `dist` 目录。

### 2. 开始下载
1. **登录喜马拉雅**：在浏览器中登录您的喜马拉雅账号。
2. **选择专辑**：进入您想要下载的专辑详情页（确保已购买）。
3. **设置本地文件夹（可选但强烈建议）**：
   - 点击页面右下角的 **“本地列表”** 按钮。
   - 选择您存放该专辑音频的本地文件夹。这能让插件识别哪些文件已存在。
4. **开始下载**：
   - 点击 **“下载全部”** 开始自动抓取并下载。
   - 或者点击 **“下载本页”** 仅下载当前可见的章节。

## 开发者说明

本项目使用 Vite + React 开发。

### 常用命令
- `npm run dev`：启动开发服务器。
- `npm run build`：生成生产环境构建到 `dist` 目录。
